### Prerequisites

- npm
  ```sh
  npm install npm@latest -g
  ```

<br>

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/fenish/EasyTube.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```

## Usage

1. Run dev server

   - Yarn
     ```sh
     yarn dev
     ```
   - Npm
     ```sh
     npm run dev
     ```

2. Build project

   - Yarn
     ```sh
     yarn build
     ```
   - Npm
     ```sh
     npm run build
     ```
